# CfEditor 编辑器功能说明

## 🎯 核心功能

### 代码编辑
- **语法高亮**: 支持C++语法高亮，包括关键词、类型、注释、字符串等
- **行号显示**: 可切换显示/隐藏行号
- **字体大小调节**: 支持10-32号字体大小调节
- **自动保存**: 防抖自动保存，避免数据丢失

### 文件管理
- **文件创建**: 支持创建新的.cpp文件
- **文件打开**: 快速打开已有文件
- **文件重命名**: 支持文件重命名
- **文件删除**: 安全删除文件
- **文件列表**: 按修改时间排序的文件列表

## 🚀 增强功能

### 代码片段
- **快速插入**: 一键插入常用代码片段
- **内置片段**: 
  - 快速输入输出优化
  - 常用头文件
  - 循环模板
  - 向量操作
  - 快速排序
  - 二分查找
  - 优先队列
  - 集合操作

### 搜索替换
- **文本搜索**: 支持大小写不敏感搜索
- **替换功能**: 单个替换和全部替换
- **结果导航**: 在搜索结果间快速跳转

### 代码格式化
- **自动缩进**: 智能缩进管理
- **空格规范**: 操作符前后自动添加空格
- **代码清理**: 清理多余空格和换行

### 撤销重做
- **编辑历史**: 支持最多50步撤销
- **智能管理**: 自动管理撤销/重做历史
- **状态保护**: 避免撤销操作被记录

## ⌨️ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd + S` | 保存文件 |
| `Cmd + F` | 打开搜索 |
| `Cmd + Z` | 撤销 |
| `Cmd + Y` | 重做 |
| `Cmd + R` | 格式化代码 |

## 🎨 界面特性

### 工具栏
- **文件管理**: 文件夹图标，快速访问文件
- **字体控制**: +/- 按钮调节字体大小
- **代码片段**: 橙色图标，插入代码片段
- **搜索功能**: 绿色图标，打开搜索面板
- **代码格式化**: 紫色图标，格式化代码
- **撤销重做**: 蓝色箭头，编辑历史管理
- **运行按钮**: 播放图标，预留运行功能

### 面板设计
- **文件浏览器**: 中等/大尺寸面板
- **代码片段**: 紧凑的片段列表
- **搜索替换**: 简洁的搜索界面

## 🔧 技术特性

### 性能优化
- **防抖高亮**: 避免频繁重高亮影响性能
- **异步保存**: 后台保存，不阻塞UI
- **内存管理**: 限制撤销历史数量

### 用户体验
- **拖拽收起**: 向下拖拽收起键盘
- **焦点管理**: 智能焦点切换
- **错误提示**: 友好的错误信息显示

## 📱 移动端适配

- **触摸友好**: 优化的触摸交互
- **键盘适配**: 智能键盘管理
- **响应式布局**: 适配不同屏幕尺寸

## 🎯 竞赛编程专用

- **C++优化**: 专为C++竞赛编程设计
- **快速模板**: 内置常用竞赛模板
- **代码片段**: 竞赛常用代码片段
- **文件扩展名**: 支持.cpp, .cc, .cxx, .c, .h, .hpp, .hxx

这个编辑器专为Codeforces等竞赛编程平台设计，提供了完整的代码编辑、管理和优化功能。
